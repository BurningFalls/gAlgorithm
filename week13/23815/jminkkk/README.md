## 💡 풀이 방법 요약

+ 두 조합 중 * 또는 + 를 사용
  + ** or ++ 이면 뒤의 값이 큰 값
  + 두 조합이 둘 다라면, 계산해보고 큰 값 <- 그리디로
+ 광고: 두 조합이 / 또는 -가 있을 경우
  + 이런 조합이 여러개 라면? 
  + 값을 2개 저장 
    + 이번에 써버린 값
      + 안 쓴 값과 이전에 써버린 값 비교해서 어떤 게 큰지
    + 안 쓰기
      + 안쓴값에 이번 것 계산
    
## 👀 실패 이유
1. 안 쓰면 - 가 되는 턴에서 즉 무조건 써야 되는 거 체크 안해서
2. 안 쓴거에 광고 써서 쓴 것보다 작다면 쓴 거 갱신 (skip)을 잘못함
