# Info
[25571 지그재그 부분배열](https://boj.kr/25571)

## 💡 풀이 방법 요약
연속되는 지그재그 부분으로 분할해서 부분 안에서 공식써서 합산?
ㄴ 이라고 메모를 해 놨다가, 상세 구현이 잘 안 떠올라서 가장 마지막에 풀었다.

저 '공식' 이라는 것은 생각해 보면 `i < j` 를 만족하는 구간 내의 모든 순서쌍이므로 길이가 `l`일 때 `l(l+1)/2` 이다.

이제 연속되는 지그재그 부분으로 분할만 하면 되는데, 투포인터를 사용하면 배열을 처음부터 끝까지 한 번만 훑으면 된다.

늘 그렇듯 `[left, right)` 로 잡는다.  
1. 두 포인터가 맞닿으면 `right` 포인터를 전진시킨다.
2. 두 포인터가 한 칸 차이날 때(`left+1 == right`)는 두 값이 같지만 않으면 항상 지그재그이다.
   - 배열의 두 값이 같으면 `left` 와 `right`를 함께 전진한다.
   - 두 값이 다르면 `right`만 전진한다.
3. 그 외의 모든 경우
   - 현재 포인터가 이루는 부분 배열이 지그재그 배열인지를 검증하여 맞다면 `right`를 전진한다.
   - 아니라면 해당 부분 배열은 거기서 종료되어야 하므로 `left`를 `right-1`까지 한 번에 당겨 준다.
     - 이후 '공식'을 사용해 현재 부분 배열에서의 페어 개수를 구하고 합산한다.
